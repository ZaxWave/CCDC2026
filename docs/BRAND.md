# LightScan Brand

- Primary mark: `src/frontend/staticfiles/lightscan-mark.svg`
- Horizontal lockup: `src/frontend/staticfiles/lightscan-lockup.svg`
- Favicon: `src/frontend/staticfiles/favicon.svg`
- PWA icons: `src/frontend/staticfiles/icon-192.svg`, `src/frontend/staticfiles/icon-512.svg`

## Usage

- Use `lightscan-mark.svg` for compact surfaces, app icons, favicons, avatars, and loading states.
- Use `lightscan-lockup.svg` for navigation, login headers, splash screens, posters, docs covers, and wide headers.
- The current identity is text-led: italic `LightScan` for the wordmark and centered italic `Ls` for compact icons.
- Do not recreate the logo with decorative circles, scan beams, or ad hoc graphics. Reference the SVG assets directly where the platform supports SVG.

## Meaning

The symbol combines a diagonal road segment, a scan aperture, and a detected pavement crack. It is designed to communicate road inspection, machine vision, and defect recognition without adding decorative clutter.
